# Picasso Design Brief

## Product

热龄卫士：面向街道、社区与卫健部门的城市热健康风险预警与避险服务调度平台。

## Design Direction

- **Archetype**: Industrial / Utilitarian Civic Command Center
- **Intent**: 像“城市运行指挥界面”，而不是普通课程作业大屏
- **Differentiator**: 危险信号条、状态舱面板、实验结论前置、强对比数据标签

## Visual Commitments

- **Display Font**: `Archivo`
- **Body Font**: `IBM Plex Sans`
- **Mono Font**: `JetBrains Mono`
- **Accent**: 冷蓝 + 琥珀 + 信号绿
- **Layout**: 左主叙事 + 右决策舱，非居中，强调层级和状态感
- **Motion**: 轻量进入动效与悬停抬升，避免夸张炫技

## Explicit Rejections

- 拒绝居中堆叠式 SaaS 首页
- 拒绝统一卡片栅格的模板感
- 拒绝单一蓝紫渐变和默认字体

## Radius Strategy

- Hero: `30px`
- Panel: `24px`
- Card: `20px`
- Soft blocks: `14px`
- Pills / chips: `999px`

## UI Rules

1. 关键实验结论必须直接上首页
2. 图表必须采用深色背景下的高对比风格
3. 推荐点位表必须显示“新增覆盖”和“改善时间”双重价值
4. 所有说明性文案优先服务“决策”和“调度”
